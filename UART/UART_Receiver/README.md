# UART Packet Analyzer

## Overview
A UART packet analyzer built using the ESP32 that receives serial data, analyzes packets, and displays important information on both the Serial Monitor and an SH1106 OLED display.

## Features
- UART packet reception
- Packet counter
- Packet length calculation
- Total bytes received
- Average packet length
- Longest packet detection
- Shortest packet detection
- ASCII table
- HEX representation
- Binary representation
- Live OLED dashboard

## Hardware Used
- ESP32 Dev Module
- SH1106 OLED Display

## Libraries
- Wire
- Adafruit_GFX
- Adafruit_SH110X
