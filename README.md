# Multi-Protocol Embedded Communication Suite

A collection of embedded communication projects demonstrating the practical implementation of major serial communication protocols using ESP32 and Arduino Uno.

## Protocols Included

- UART Communication
- I²C Master-Slave Network
- CAN & SPI Telemetry Network
- RS485 Sensor Network

## Technologies

- ESP32
- Arduino Uno
- UART
- SPI
- I²C
- CAN Bus
- RS485
- Embedded C/C++
- Arduino IDE

Each project contains:
- Source Code
- Hardware Connections
- Images
- Project Documentation
